Content delivery: Social networking

Brand trust

  • Use consistent images and colours
  • Use consistent words and phrases
  • Keep messages simple

Facebook

  • 1.7 billion monthly users
  • 1.3 billion daily users
  • 4.5 billion likes generated daily

Facebook accessibility features

  • Keyboard shortcuts
  • Automatic alt texts

Facebook content

  • Use captions to describe images
  • Do not use text in images
  • Use plain and simple language

Facebook alternatives

  • m.facebook.com
  • Native apps

YouTube

  • 1 billion daily users
  • 4 billion videos watched daily
  • 6 billion hours of video watched monthly

YouTube accessibility features

  • Keyboard shortcuts
  • Autocaps

YouTube content

  • Provide a transcript
  • Provide audio description

YouTube alternatives

  • Native apps

Twitter

  • 313 million daily users
  • 500 million daily tweets
  • 350 thousand tweets every minute

Twitter accessibility features

  • Keyboard shortcuts
  • Alt text for images

Twitter content

  • Use proper words and avoid acronyms
  • Capitalise the first letter of words in #HashTags
  • Provide alt texts

Twitter alternatives

  • Easy Chirp
  • Native apps
  • Desktop apps
